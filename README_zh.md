# ssher

简体中文 | [English](README.md)

ssher 是一款连接远程服务器命令行工具，简单易用。

## 安装

```bash
cargo install ssher
```

## 使用

1. 选择服务器并连接

```bash
ssher
```

2. 添加服务器

```bash
ssher add
```

3. 删除服务器

```bash
ssher rm
```

4. 查看服务器

```bash
ssher ls
```

5. 重命名

```bash
ssher rename
```

6. 查看版本

```bash
ssher version
```

7. 查看帮助

```bash
ssher help
```

## 配置

配置文件保存在 `~/.ssher.yaml` 文件中。
