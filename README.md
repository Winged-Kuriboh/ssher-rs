# ssher

English | [简体中文](./README_zh.md)

ssher is an easy-to-use command line tool for connecting to remote servers.

## Installation

```bash
cargo install ssher
```

## Usage

1. Select a server and connect

```bash
ssher
```

2. Add a server

```bash
ssher add
```

3. Remove a server

```bash
ssher rm
```

4. List servers

```bash
ssher ls
```

5. Rename a server

```bash
ssher rename
```

6. Check version

```bash
ssher version
```

7. Help

```bash
ssher help
```

## Configuration

The configuration file is saved in the `~/.ssher.yaml` file.
